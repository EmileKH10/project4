# 🎓 EXAMIX – מערכת לערבול מבחנים

EXAMIX היא מערכת ווב רספונסיבית המאפשרת למורים ולסטודנטים ליצור מספר גרסאות למבחן מתוך קובץ PDF אחד.

המערכת כוללת:
- העלאת טופס PDF
- ערבול שאלות ותשובות
- יצירת מספר גרסאות
- שמירת נתונים בבסיס נתונים
- צפייה בהיסטוריית מבחנים
- מערכת התחברות והרשמה

המערכת נבנתה באמצעות:
HTML5, CSS3, JavaScript, jQuery, PHP ו-MySQL.
